What is this?

This is a chrome extension that allows you to create bookmarks lists.

    How to start?
  • Type a new list title using the title box
  • Click the button "Create List" to create your new list

    Now your new list will appear within the available lists to select

  • Select a list
  • Type a title for your item inside the title box
  • Click "Add to list" to add the item to your selected list

    Now your item will be listed within your selected list

    Deleting lists:
  • Select a list from the dropdown menu
  • Click the button "Remove list" to delete the list
    Deleting items from lists:
  • Select the list with the item that you wish to delete
  • On the right side of the item title you will find a "x" icon
  • Click the "x" icon to delete the desired item

Do my lists vanish if I close my browser?

No. Your lists are saved within your own Chrome Cloud services. Bear in mind that there is a limit of 100kb per user

Do I need to be logged into Chrome to use it?

Yes. Since all of your lists are saved in your own cloud, you need to be logged in to use this extension
